# HomeStay-Dorm-System

Đồ án TH môn Phân Tích Thiết Kế Hệ Thống Thông Tin (PTTK)

## Giới thiệu

HomeStay-Dorm-System là một hệ thống quản lý homestay và ký túc xá, được xây dựng sử dụng Handlebars (template engine), JavaScript và CSS.

## Yêu cầu hệ thống

- Node.js (Nên sử dụng phiên bản mới nhất)
- npm (Node Package Manager)

## Hướng dẫn cài đặt và chạy dự án

1. **Clone repository về máy:**
   ```bash
   git clone https://github.com/quangson23082005-create/HomeStay-Dorm-System.git
   ```

2. **Truy cập vào thư mục dự án:**
   ```bash
   cd HomeStay-Dorm-System
   ```

3. **Cài đặt các package phụ thuộc:**
   ```bash
   npm install
   ```

4. **Khởi động ứng dụng:**
   ```bash
   npm start
   ```

5. **Truy cập ứng dụng:**
    - Mở trình duyệt và truy cập địa chỉ (theo mặc định):
      ```
      http://localhost:3000
      ```
    - (Nếu port mặc định đã thay đổi, hãy kiểm tra thông báo terminal khi chạy lệnh `npm start`.)

## Thông tin thêm

- Template sử dụng: Handlebars
- Ngôn ngữ lập trình: JavaScript, CSS

Nếu gặp khó khăn khi cài đặt hoặc sử dụng ứng dụng, vui lòng tạo issue mới trên repository này hoặc liên hệ với nhóm phát triển.
